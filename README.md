# automation-watchdog

Real-time health monitoring for AI automation workflows (n8n, Make, Zapier). Detects silent failures before they corrupt data.

## What it does

Checks workflow health signals and raises alerts when pipelines degrade. Built for automation-heavy stacks where one missed run can cascade into bad data or missed actions.

## Tech stack

- Python
- Automation integrations: n8n, Make, Zapier

## Get started

1. Install dependencies
2. Configure your workflow sources
3. Start monitoring and alerting

## Use case

Use this when you rely on scheduled automations and need a lightweight guardrail against silent breakage.
